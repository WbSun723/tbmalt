classmethod load_data(path, sources, targets, pbc=False, device=None)[source]

Load a collection of data-points into a dataset instance.

Parameters:

  • path (str) – path to the database.

  • sources (List[str]) – a list of paths specifying the groups from which data should be loaded; one for each system.

  • targets (Union[List[str], Dict[str, str]]) – paths relative to source specifying the HDF5 datasets to load. Results are loaded from each source group, packed together and then placed in the class’s data attribute. The key under which the data is stored can be specified using a dictionary of the form {name: path/to/dataset}, if a list is provided then the path is used as the key.

  • pbc (bool) – Whether read cells.

  • device (Optional[device]) – Device on which to create any new tensors. [DEFAULT=None]

Return type:

DataSetIM

Returns:

dataset – a dataset object holding the requested data.

Notes

Structural information must be present in the group specified by source. Specifically, an attempt will be made to instantiate a Geometry instance from a sub-group of the name “geometry”, failing this it will be assumed that the required date is present in the main group itself, i.e. source.

classmethod load_data_batch(path, source, targets, pbc=False, device=None)[source]

Load a batch of data from an HDF5 database into a dataset instance.

Unlike load_data, this method assumes that the data stored within the database at the location specified by source is pre-batched.

Parameters:

  • path (str) – path to the HDF5 database.

  • source (str) – path specifying the group within the database from which data should be loaded.

  • targets (Union[List[str], Dict[str, str]]) – paths relative to source specifying the HDF5 datasets to load. Results are placed in the class’s data attribute. The key under which the data is stored can be specified using a dictionary of the form {name: path/to/dataset}, if a list is provided then the path is used as the key.

  • pbc (bool) – Whether read cells.

  • device (Optional[device]) – Device on which to create any new tensors. [DEFAULT=None]

Return type:

DataSetIM

Returns:

dataset – a dataset object holding the requested data.

Notes

Structural information must be present in the group specified by source. Specifically, an attempt will be made to instantiate a Geometry instance from a sub-group of the name “geometry”, failing this it will be assumed that the required date is present in the main group itself, i.e. source.

Each of the datasets loaded is assumed to be a packed array with with a data-point for each entry in the batch. That is to say if there are 100 molecules there should be 100 sets of, for example, charges. Furthermore, datasets are assumed to batched along the the first dimension.
