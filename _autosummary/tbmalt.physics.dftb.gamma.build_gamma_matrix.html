2.1.5.1.6.1. build_gamma_matrix

build_gamma_matrix(geometry, orbs, invr, hubbard_Us, scheme='exponential')[source]

Construct the gamma matrix

Parameters:

  • geometry (Geometry) – Geometry object of the system(s) whose gamma matrix is to be constructed.

  • orbs (OrbitalInfo) – OrbitalInfo instance associated with the target system.

  • invr (Tensor) – 1/R matrix.

  • hubbard_Us (Tensor) – Hubbard U values. one value should be specified for each atom or shell depending if the calculation being performed is atom or shell resolved.

  • scheme (Union[Literal['exponential', 'gaussian'], str]) – scheme used to construct the gamma matrix. This may be either either “exponential” or “gaussian”. [DEFAULT=”exponential”]

Return type:

Tensor

Returns:

gamma_matrix – the resulting gamma matrix.

Examples

>>> from tbmalt import OrbitalInfo, Geometry
>>> from tbmalt.physics.dftb.gamma import build_gamma_matrix
>>> from ase.build import molecule
>>>
>>> # Preparation of system to calculate
>>> geo = Geometry.from_ase_atoms(molecule('CH4'))
>>> orbs = OrbitalInfo(geo.atomic_numbers,
...                    shell_dict= {1: [0], 6: [0, 1]})
>>> hubbard_U = torch.tensor([0.3647, 0.4196, 0.4196, 0.4196, 0.4196])
>>> r = geo.distances
>>> r[r != 0.0] = 1.0 / r[r != 0.0]
>>> # Build the gamma matrix
>>> gamma = build_gamma_matrix(geo, orbs, r, hubbard_U, 'exponential')
>>> print(gamma)
tensor([[0.3647, 0.3234, 0.3234, 0.3234, 0.3234],
        [0.3234, 0.4196, 0.2654, 0.2654, 0.2654],
        [0.3234, 0.2654, 0.4196, 0.2654, 0.2654],
        [0.3234, 0.2654, 0.2654, 0.4196, 0.2654],
        [0.3234, 0.2654, 0.2654, 0.2654, 0.4196]])
