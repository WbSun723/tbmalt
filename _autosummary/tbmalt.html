2.1. tbmaltÔÉÅ

The top-level directory module of the TBMaLT package.

This toolkit is intended to aid in the prototyping, development and training of new tight-binding level machine-leaning methods.

Modules

common

A collection of useful code abstractions.

data

Constants & reference data required by TBMaLT is stored in the data module.

io

This module contains all IO specific components of the TBMaLT code-base.

ml

This module houses the majority of TBMaLT's machine learning code.

physics

Code pertaining to physical modes, methods, and their properties.

structures

A module for holding data structures and any associated code.

tools

A collection of miscellaneous helper modules.
