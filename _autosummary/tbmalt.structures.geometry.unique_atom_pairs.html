2.1.6.1.4. unique_atom_pairs

unique_atom_pairs(geometry, return_ordered_pairs=False)[source]

Returns a tensor specifying all unique atom pairs.

This takes Geometry instance and identifies all atom pairs. This use useful for identifying all possible two body interactions possible within a given system.

Parameters:

  • geometry (Geometry) – Geometry instance representing the target system.

  • return_ordered_pairs (bool) – bool indicating whether to consider pairs as ordered. If set to False, pairs like (1, 2) and (2, 1) are treated as equivalent and only the former is returned. If set to True, both (2, 1) and (1, 2) will be returned as they will be considered to be distinct. [DEFAULT=False]

Return type:

Tensor

Returns:

unique_atom_pairs – A tensor specifying all unique atom pairs.
