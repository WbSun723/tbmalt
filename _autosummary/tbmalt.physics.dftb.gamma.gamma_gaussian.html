2.1.5.1.6.4. gamma_gaussian

gamma_gaussian(geometry, orbs, hubbard_Us)[source]

Constructs the gamma matrix via the Gaussian method.

Elements of the gamma matrix are calculated via the following equation:

\[\gamma_{ij}(R_{ij}) = \frac{\text{erf}(C_{ij}R_{ij})}{R_{ij}}\]

Where the coefficients \(C\) are calculated as

\[C_{ij} = \sqrt{\frac{4\ln{2}}{\text{FWHM}_{i}^2 + \text{FWHM}_{j}^2}}\]

and the full width half max values, FWHM, like so

\[\text{FWHM}_{i} = \sqrt{\frac{8\ln{2}}{\pi}}\frac{1}{U_i}\]

Where U and R are the hubbard U and distance values respectively.

Parameters:

  • geometry (Geometry) – Geometry object of the system whose gamma matrix is to be constructed.

  • orbs (OrbitalInfo) – OrbitalInfo instance associated with the target system.

  • hubbard_Us (Tensor) – Hubbard U values. one value should be specified for each atom or shell depending if the calculation being performed is atom or shell resolved.

Return type:

Tensor

Returns:

gamma – gamma matrix.

Notes

One Hubbard U value must be specified for each atom or each shell based on whether an atom or shell resolved calculation is being performed. Note that this must be consistent with the shell_resolved attribute of the supplied OrbitalInfo instance orbs.

Currently, the Hubbard U values must be specified manually, however the option to supply a feed object will be added at a later data. This will facilitate the automated construction of the Hubbard U tensor and shall permit the use of environmentally dependent U values if desired.

Examples

>>> from tbmalt import OrbitalInfo, Geometry
>>> from tbmalt.physics.dftb.gamma import gamma_gaussian
>>> from ase.build import molecule
>>>
>>> # Preparation of system to calculate
>>> geo = Geometry.from_ase_atoms(molecule('CH4'))
>>> orbs = OrbitalInfo(geo.atomic_numbers,
...                    shell_dict= {1: [0], 6: [0, 1]})
>>> hubbard_U = torch.tensor([0.3647, 0.4196, 0.4196, 0.4196, 0.4196])
>>> # Build the gamma matrix
>>> gamma = gamma_gaussian(geo, orbs, hubbard_U)
>>> print(gamma)
tensor([[0.3647, 0.3326, 0.3326, 0.3326, 0.3326],
        [0.3326, 0.4196, 0.2745, 0.2745, 0.2745],
        [0.3326, 0.2745, 0.4196, 0.2745, 0.2745],
        [0.3326, 0.2745, 0.2745, 0.4196, 0.2745],
        [0.3326, 0.2745, 0.2745, 0.2745, 0.4196]])
