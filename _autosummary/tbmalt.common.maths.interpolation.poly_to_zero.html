2.1.1.6.10.2. poly_to_zero

poly_to_zero(xx, dx, inv_x, y0, y0p, y0pp)[source]

Get interpolation if beyond the grid range with 5th order polynomial.

Parameters:

  • xx (Tensor) – Grid points.

  • dx (Tensor) – The grid point range for y0 and its derivative.

  • inv_x (Tensor) – Reciprocal of dx

  • y0 (Tensor) – Values to be interpolated at each grid point.

  • y0p (Tensor) – First derivative of y0.

  • y0pp (Tensor) – Second derivative of y0.

Return type:

Tensor

Returns:

yy – The interpolation values with given xx points in the tail.

Notes

The function poly_to_zero realize the interpolation of the points beyond the range of grid points, which make the polynomial values smoothly converge to zero at the boundary. The variable dx determines the point to be zero. This code is consistent with the function poly5ToZero in DFTB+.
