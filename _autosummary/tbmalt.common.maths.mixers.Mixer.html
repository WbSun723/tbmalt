2.1.1.6.11.2. Mixer

class Mixer(is_batch, tolerance=1e-06)[source]

Bases: ABC

This is the abstract base class upon which all mixers are to be based.

This abstract base class provides the template on which all mixers are to be built.

Parameters:

  • is_batch (bool) – Set to true when mixing a batch of systems, False when not.

  • tolerance (float) – Max permitted difference in between successive iterations for a system to be considered “converged”. [DEFAULT=1E-6]

Attributes and Properties

converged

Tensor of bools indicating convergence status of the system(s).

delta

Difference between the current & previous systems

step_number

Integer tracking the number of mixing iterations performed.

Methods

abstractmethod cull(cull_list, new_size=None)[source]

Purge select systems form the mixer.

This is useful when a subset of systems have converged during mixing.

Parameters:

  • cull_list (Tensor) – Tensor with booleans indicating which systems should be culled (True) and which should remain (False).

  • new_size (Union[Size, List[int], None]) – New anticipated size of future inputs excluding the batch dimension. This is used to allow superfluous padding values to be removed form subsequent inputs.

abstractmethod reset()[source]

Resets the mixer to its initial state.
