2.1.1.6.10.4. CubicSpline

class CubicSpline(x, y, tail=1.0)[source]

Bases: Feed

Cupic spline interpolator.

An entity for piecewise interpolation of data via a cupic polynomial spline which is twice continuously differentiable.

Parameters:

  • x (Tensor) – A one dimensional tensor specifying the interpolation grid points, i.e. the knot locations.

  • y (Parameter) – Interpolation values, i.e. the knot values, associated with each grid point. For single series interpolation this should be an array of length “n”, where “n” is the number of grid points present in x. For batch interpolation this should be an “m” by “n” tensor. Note that this must be a Parameter rather than Tensor instance.

  • tail (Real) – Distance over which to smooth the tail.

Keyword Arguments:

coefficients – 0th, 1st, 2nd and 3rd order parameters in cubic spline.

Examples

>>> from tbmalt.common.maths.interpolation import CubicSpline
>>> import torch
>>> from torch.nn import Parameter
>>> x = torch.linspace(1, 10, 10)
>>> y = Parameter(torch.sin(x), requires_grad=False)
>>> spline = CubicSpline(x, y)
>>> spline(torch.tensor([3.5]))
tensor([-0.3526])
>>> torch.sin(torch.tensor([3.5]))
tensor([-0.3508])
